# 🧴 Perfume Inventory System

A lightweight web application built with Python, Flask, and SQLite to manage stock, calculate total inventory value, and track perfume sales.

## Features
- 🔒 **User Authentication:** Admin login with hashed password protection.
- 📦 **Inventory Management:** Add, update, and delete fragrance stock items.
- 💰 **Total Stock Calculation:** Dynamic sum of all stock values.
- 🔍 **Search & Filter:** Easily filter stock by item names.

## Tech Stack
- **Backend:** Python (Flask)
- **Database:** SQLite3
- **Frontend:** HTML5, CSS3, Jinja2 Templates
